# eBay Cassini Diagnostic Tool

**Fix zero views and failing sales on eBay**

A free, open-source tool built for part-time and retiring sellers who are being buried by Cassini.

### Features
- New Listing Optimizer with live Cassini scoring
- Portfolio Analyzer — upload your Active Listings CSV and get prioritized fixes
- Clear guidance on when to Revise vs Sell Similar

### How to Run (Super Simple)

1. Install Python (if you don't have it)
2. Open terminal / command prompt
3. Run:
   ```bash
   pip install streamlit pandas
   streamlit run app.py
