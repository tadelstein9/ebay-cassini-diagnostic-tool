# Bynari Insight

Onboarding tool for new eBay sellers. Walks you through your first compliant
listing — category, item specifics, photos, condition, title, description —
with the rules explained as you go.

Built by [Bynari](https://bynari-insight.com).

## Status

Early deployment. The app shell and privacy page are live; the onboarding
flow ships in stages.

## For developers

Local run:

    pip install -r requirements.txt
    streamlit run app.py

Privacy page: append `?page=privacy` to the root URL.
